# 🚀 AI Hiring Copilot

AI Hiring Copilot is an end-to-end AI-powered recruitment assistant that helps candidates optimize their job search journey. Simply upload your resume and let the system analyze your profile, identify skill gaps, recommend jobs, generate interview preparation content, and answer career-related questions through an intelligent chatbot.

---

## ✨ Features

### 📄 Resume Analysis
- Upload resumes in PDF or DOCX format
- Extract candidate information automatically
- Identify technical and soft skills
- Generate structured candidate profiles
- Resume scoring and feedback

### 🤖 AI Chat Assistant
- Career guidance and mentorship
- Resume improvement suggestions
- Job application assistance
- Technical concept explanations
- Personalized career recommendations

### 🎯 Job Matcher
- Matches resume skills with available job opportunities
- Calculates skill compatibility score
- Highlights missing skills
- Displays relevant positions with:
  - Company Name
  - Location
  - Salary Range
  - Match Percentage
  - Required Skills

### 🎤 Interview Preparation
- Generates role-specific interview questions
- Technical interview practice
- Behavioral interview questions
- HR interview preparation
- Personalized mock interview experience

### 📊 Skill Gap Analysis
- Identifies missing skills for target roles
- Recommends learning paths
- Suggests certifications and courses
- Tracks improvement areas

### 📝 Resume Builder & Optimization
- AI-generated resume enhancements
- ATS-friendly recommendations
- Keyword optimization
- Professional formatting suggestions

### 📈 Career Insights
- Industry trends
- Market demand analysis
- Salary insights
- Emerging technology recommendations

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/your-username/AIRecruiterAgent.git
cd AIRecruiterAgent
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Environment

Windows:

```bash
venv\Scripts\activate
```

Linux/Mac:

```bash
source venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Configure Environment Variables

Create a `.env` file:

```env
GOOGLE_API_KEY=your_api_key_here
GEMINI_API_KEY=your_api_key_here
```

### Run Application

```bash
python ai_recruiter_agent.py
```
